A high-security, cross-platform encryption utility built with **Flutter** and **OpenPGP**. This application allows users to perform symmetric PGP encryption and decryption with a focus on privacy and a clean, modern Dark Mode interface.

## ‚ú® Features

- **Symmetric Encryption**: Secure your text using robust PGP standards with a password.
- **Cross-Platform**: Designed to run seamlessly on **Android** and **Ubuntu Linux**.
- **Dark Mode UI**: A consistent, eye-friendly dark theme across all platforms.
- **Copy to Clipboard**: One-tap functionality to copy PGP blocks for easy sharing.
- **Privacy First**: No data is sent to any server; all encryption happens locally on your device.
